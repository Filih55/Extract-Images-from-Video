Steps to extract images from video:

1. Install OpenCV first:
   pip install opencv-python

2. Choose extraction option:
   - perdetik.py: Extract per second
   - permenit.py: Extract per minute
   - campuran.py: Combination of per second and per minute (remove # to select)
   - main.py: Extract per frame (caution, long videos will produce many images)

3. Select video file:
   Change the directory address and video file name in the code, example:
   F:\RR\Video\tiktok.mp4

Note: The longer the video, the more images will be extracted.

# Ekstraksi Gambar dari Video

Program ini memungkinkan Anda untuk mengekstrak gambar dari file video menggunakan OpenCV.

## Persyaratan

Instal OpenCV terlebih dahulu:

```
pip install opencv-python
```

## Penggunaan

Program ini menyediakan beberapa opsi ekstraksi:

1. `perdetik.py` - Ekstrak gambar per detik
2. `permenit.py` - Ekstrak gambar per menit  
3. `campuran.py` - Kombinasi ekstraksi per detik dan per menit (hapus tanda # untuk memilih)
4. `main.py` - Ekstrak gambar per frame (hati-hati, video panjang akan menghasilkan banyak gambar)

## Memilih File Video

Untuk memilih file video, ubah alamat direktori dan nama file di dalam kode. Contoh:

```python
video_path = "F:\RR\Video\tiktok.mp4"
```
